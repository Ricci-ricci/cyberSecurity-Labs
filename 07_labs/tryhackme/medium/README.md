# Medium Difficulty - TryHackMe Writeups

This directory contains writeups for **Medium** difficulty boxes from TryHackMe.

## About Medium Boxes

Medium boxes are designed for intermediate users and typically cover:
- Advanced enumeration techniques
- Less common vulnerabilities
- Multiple exploitation paths
- Complex privilege escalation scenarios
- Real-world attack scenarios

## Writeup Format

Each writeup should follow this structure:
1. **Reconnaissance** - Comprehensive port scanning and service enumeration
2. **Enumeration** - Deep dive into services and web applications
3. **Initial Foothold** - Vulnerability discovery and exploitation
4. **Lateral Movement** - User pivoting (if applicable)
5. **Privilege Escalation** - Advanced techniques to gain root/admin access
6. **Flags** - User and root flags
