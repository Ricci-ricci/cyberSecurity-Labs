# Easy Difficulty - TryHackMe Writeups

This directory contains writeups for **Easy** difficulty boxes from TryHackMe.

## About Easy Boxes

Easy boxes are designed for beginners and typically cover:
- Basic enumeration techniques
- Common vulnerabilities
- Straightforward exploitation paths
- Fundamental privilege escalation methods

## Writeup Format

Each writeup should follow this structure:
1. **Reconnaissance** - Port scanning and service enumeration
2. **Initial Foothold** - Finding and exploiting vulnerabilities
3. **Privilege Escalation** - Gaining root/admin access
4. **Flags** - User and root flags
