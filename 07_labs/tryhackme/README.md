## Writeup Guidelines

Each writeup follows a structured format:
1. **Box Information** - Name, difficulty, platform
2. **Reconnaissance** - Initial scanning and enumeration
3. **Exploitation** - Vulnerability discovery and exploitation
4. **Privilege Escalation** - Path to root/admin access
5. **Flags** - User and root flags captured
6. **Lessons Learned** - Key takeaways and techniques
