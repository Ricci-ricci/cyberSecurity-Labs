# Hard Difficulty - TryHackMe Writeups

This directory contains writeups for **Hard** difficulty boxes from TryHackMe.

## About Hard Boxes

Hard boxes are designed for advanced users and typically cover:
- Extensive enumeration across multiple services
- Obscure or chained vulnerabilities
- Custom exploit development
- Complex privilege escalation chains
- Multiple pivot points and lateral movement
- Advanced persistence techniques
- Real-world enterprise scenarios

## Writeup Format

Each writeup should follow this structure:
1. **Reconnaissance** - Thorough port scanning and service enumeration
2. **Enumeration** - Deep analysis of all services and applications
3. **Initial Foothold** - Complex vulnerability discovery and exploitation
4. **Lateral Movement** - Multi-user pivoting and credential harvesting
5. **Privilege Escalation** - Advanced techniques and exploit chains
6. **Post-Exploitation** - Persistence, data exfiltration, and cleanup
7. **Flags** - User and root flags
8. **Lessons Learned** - Key takeaways and techniques
